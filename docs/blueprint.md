## Style Guidelines:

- Primary color: Electric Blue (#2563EB) to inspire trust.
- Background color: Light-blue (#F0F9FF), almost white, derived from the primary.
- Accent color: Orange (#F97316) for CTAs to draw attention.
- Font: 'Poppins' (sans-serif) for headlines and body, to convey a modern and accessible feel.
- Mobile-first design optimized for 99% mobile traffic with large, easy-to-tap buttons and a sticky footer for key CTAs.
- Sharp, modern icons, complementing the 'Fresh & Clear' aesthetic.
- Subtle, smooth transitions for page elements to create a high-end boutique feel.
